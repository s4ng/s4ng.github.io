# 공부 정리

강의와 책을 공부하며 정리한 노트를 모아둔 공간입니다.

왼쪽 사이드바의 계층은 `docs/` 폴더 구조에서 **자동 생성**됩니다.
새 주제를 추가하려면 `docs/` 아래에 폴더를 만들고 `_category_.json`으로
라벨과 순서를 지정하면 됩니다.

## 구조 예시

- **운영체제 강의** — 프로세스, 가상 메모리 등
  - 폴더를 중첩하면 사이드바 depth가 그만큼 깊어집니다.

자세한 작성 방법은 [Docusaurus 공식 문서](https://docusaurus.io/docs/sidebar/autogenerated)를 참고하세요.
